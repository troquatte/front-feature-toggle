Mas o que é Feature Toggle?

Feature toggle é uma estratégia que nos permite fazer integração contínua. Essa estratégia consiste em habilitar e desabilitar recursos e funcionalidades em um sistema sem alteração de código.

Mas o que é Feature Toggle?

Para os desenvolvedores que enfrentam dificuldade em desenvolver feature toggle em virtude das restrições/impedimentos ou até mesmo por não ter uma ferramenta para o mesmo, a Medusa é um hub de feature toggle que diminui significativamente o leadtime das entregas e é bem mais rápido, simples e menos burocrático para implementar comparado com a solução do Firebase, AWS, etc.
